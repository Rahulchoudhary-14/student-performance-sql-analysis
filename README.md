# 📊 Student Performance Analytics Using SQL

A comprehensive SQL project that analyzes student academic performance using **MySQL**. This project demonstrates SQL fundamentals, including database creation, data cleaning, CRUD operations, filtering, aggregation, and business-oriented data analysis to extract meaningful insights from a real-world dataset.

---

## 📌 Project Overview

This project explores a dataset containing students' academic scores and demographic information. Using SQL, the data is cleaned, transformed, and analyzed to answer analytical questions and identify trends affecting student performance.

The objective of this project is to strengthen SQL skills while showcasing practical data analysis techniques commonly used by Data Analysts and Business Analysts.

---

## 🎯 Objectives

- Design and manage a relational database using MySQL.
- Perform data cleaning and preprocessing.
- Execute analytical SQL queries.
- Generate actionable insights from student performance data.
- Practice real-world SQL operations for portfolio development.

## 📈 Business Questions Solved

This project answers several analytical questions, including:

- What is the gender distribution of students?
- What are the average Math, Reading, and Writing scores by gender?
- How many students completed the test preparation course?
- Which parental education level is most common?
- Which race/ethnicity group has the highest number of students?
- Which students achieved perfect scores?
- Which students scored below the passing threshold?
- What are the highest and lowest scores in each subject?
- What is the distribution of meal types?

---

## 📊 Key Insights

- Dataset contains **1,000 student records**.
- Female students slightly outnumber male students.
- Most students did **not** complete the test preparation course.
- Male students achieved a higher average **Math** score.
- Female students performed better in **Reading** and **Writing**.
- **Group C** represents the largest student population.
- **High School Graduate** is the most common parental education level.
- Most students received the **Standard** lunch plan.
